WSL基础：开始

发表于2024-12-16|更新于2024-12-16|WSLWSL基础
|总字数:694|阅读时长:2分钟|浏览量:

安装

1 开启相关Window服务

  1. 【设置】-【系统】-【关于】，确保操作系统内部版本大于19041，本文使用的版本为19045.5247
  2. 【控制面板】-【程序和功能】-【启用或关闭WIndow功能】
    1. 勾选【Hyper-V】
    2. 勾选【适用于Linux的Window子系统】
    3. 勾选【虚拟机平台】

2 安装WSL以及Ubuntu

  1. 打开微软商店，搜索并安装【Windows Subsystem for Linux】
  2. 打开微软商店，搜索并安装【Ubuntu 22.04.5 LTS】，你也可以选一个其他版本
  3. 上一步安装完成后，打开安装好的【Ubuntu 22.04.5 LTS】，将显示Installing, this may take a few minutes...自动安装，随后输入账户名，输入密码，确认密码

3 确认安装版本

  1. 【Ubuntu 22.04.5 LTS】安装完成后，打开任意终端，执行wsl -l -v，会显示
    NAME STATE VERSION
    *Ubuntu 22.04 Running 2

4 切换默认环境

  1. 默认是安装在C盘的，随着虚拟机里软件越装越多，很快C盘的空间就吃紧了，而且比较坑的是，就算在虚拟机里把文件给删除了，空间也不会被释放
  2. 默认位置类似：C:\Users\xxxxxxxxxxx\AppData\Local\Packages\CanonicalGroupLimited.Ubuntu22.04LTS_79rhkp1fndgsc\LocalState\ext4.vhdx
  3. 关闭所有虚拟机：
    打开任意终端输入：wsl --shutdown
  4. 导出默认环境（Ubuntu-22.04是wsl -l -v显示的NAME）：
    打开任意终端输入：wsl --export Ubuntu-22.04 D:\WSL2Ubuntu22.04.bak
  5. 注销原来的虚拟机：
    打开任意终端输入：wsl --unregister Ubuntu-22.04
  6. 重新导入（D:\Ubuntu22.04是自定义目标文件夹，请创建）
    wsl --import Ubuntu-22.04 D:\Ubuntu22.04 D:\WSL2Ubuntu22.04.bak --version 2

5 映射本机路径

  1. ubuntu中要访问windows主系统中的文件，不用复制1份，直接用 “/mnt/盘符”，就能访问
  2. 为了方便，可以创建软链接：
    ln -s /mnt/d/document/ /home/yueyl/document
    这样就把windows主系统e盘的document目录，映射到/home/yueyl/document下了

6 安装LinuxGUI应用

  1. 使用 WSL 运行 Linux GUI 应用
  2. 更新发行版中的包
    sudo apt update
  3. 安装文件管理器（可选）
    sudo apt install nautilus
  4. 安装一个终端
    sudo apt install dolphin
    sudo apt install konsole
  5. 安装 CMake、Git 和 GCC
    sudo apt install g++ cmake git build-essential gdb
  6. 安装 Vim
    sudo apt install vim
  7. 安装谷歌浏览器
    1. 将目录更改为 temp 文件夹：cd /tmp
    2. 使用 wget 下载它：wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
    3. 安装包：sudo apt install --fix-missing ./google-chrome-stable_current_amd64.deb
    4. 运行谷歌浏览器：google-chrome
    5. 如果报错：zygote_host_impl_linux.cc(101)] Running as root without –no-sandbox is not supported. See https://crbug.com/638180.
      1. 那么vim /usr/bin/google-chrome
      2. 将exec -a "$0" "$HERE/chrome" "$@"改为exec -a "$0" "$HERE/chrome" "$@" --user-data-dir --no-sandbox
      3. 以后只要google-chrome就可以打开Chrome了
