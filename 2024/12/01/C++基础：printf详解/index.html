C++基础：printf详解

发表于2024-12-01|更新于2024-12-01|C++C++基础
|总字数:289|阅读时长:1分钟|浏览量:

printf 是 C 语言标准库中的一个函数，来自于 <stdio.h> 头文件，虽然 C++ 更倾向于使用 cout，但 printf 在某些情况下仍然非常有用。

printf格式指定符

类型 printf 格式指定符
short %hd
unsigned short %hu
int %d
unsigned int %u
long %ld
unsigned long %lu
long long %lld
unsigned long long %llu
无符号整数（16进制） %x
无符号整数（8进制） %o
float %f
float（科学计数法） %e
float（自动选择%f或%e） %g
double %lf
double（科学计数法） %le
double（自动选择%f或%e） %lg
long double %Lf
long double（科学计数法） %Le
long double（自动选择%f或%e） %Lg
char（窄字符） %c
wchar_t（宽字符） %lc
bool %d
size_t %zd
size_t（十六进制） %zx
char* %s
指针 %p
百分号 %%

printf格式修饰符

格式说明符可以包含额外的标志+宽度+精度来控制输出格式：

  • 标志：- 表示左对齐，+ 在正数前加加号。
  • 宽度：指定输出字段的最小宽度。
  • 精度：用于浮点数和字符串的显示精度。

举例：%-8.3e表示打印一个浮点数，左对齐，占位宽度为8，有效数字为3
