配置API

文件位置：~/.claude/settings.json，没有的话就自己创建，有的话就增量补充

启动并使用

终端输入claude即可启动开启对话，初次启动的配置建议使用默认配置

安装 ClaudeCode for VSCode

在VSCode扩展中心搜索Claude Code for VS Code官方插件，一键安装即可
VSCode 扩展链接：Claude Code for VS Code

配置API

  1. 点击扩展商店的ClaudeCode的设置按钮“⚙”
  2. 找到Claude Code: Environment Variables并点击 settings.json 中编辑，增量添加 claudeCode.environmentVariables 如下：

重启并使用

  1. 点击扩展商店的ClaudeCode的设置按钮“⚙”
  2. 勾选 🗸 Claude Code: Use Terminal
  3. VSCode左边缘的两个ClaudeCode图标分别是单个会话和会话管理页面
  4. 右上角的ClaudeCode图标是ClaudeCode的终端（如果安装了），使用上差异不大，可能CLI更舒服
  5. 现在可以选择你习惯的方式，可以使用了

简单实用的对话技巧

  • 具体：指定文件、场景、偏好
  • 指向：“照着这个做”
  • 克制：任务分步来
  • 我想做XXXX，在动手之前，先采访我，问清楚所有你需要知道的事情。
