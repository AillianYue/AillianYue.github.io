Marp for VSCode 简介

Marp（Markdown Presentation）是一套将 Markdown 转换为演示文稿（PDF/PPTX/HTML）的开源工具链。核心哲学是用纯文本管理幻灯片，适合技术分享、组会汇报、课程讲义等场景。

核心功能：Markdown → PPT

Marp 读取 Markdown 文件，按约定的分隔符分割页面，渲染为幻灯片。一条 Markdown 对应一张幻灯片，一个 --- 分隔就是下一张。

分页规则

分隔符 效果
--- 水平分割线，前后各留空行，开始新一页
-- 当前页内分栏（需配合 CSS 自定义布局）

注意：--- 前后必须有空行，否则会被视为普通 Markdown 水平线。

自动缩放

安装

  1. 打开 VS Code，点击左侧扩展图标（或按 Ctrl+Shift+X）
  2. 搜索 Marp for VS Code（作者：marp-team）
  3. 点击安装
  4. 打开任意 .md 文件，右上角出现 “Preview” 按钮即安装成功

也可以通过命令行安装：code --install-extension marp-team.marp-vscode

使用方法

文件头（YAML Front Matter）配置

Marp 在 Markdown 的 YAML Front Matter 中读取全局配置。以下是一份推荐的标准模板：

各字段速查：

字段 取值 作用
marp true 必须，启用 Marp 渲染
theme default / uncover / gaia / 自定义 切换主题
size 16:9 / 4:3 幻灯片宽高比
paginate true / false 是否显示页码
header 字符串 全局页眉（每页顶部）
footer 字符串 全局页脚（每页底部）
class 字符串 全局幻灯片 CSS 类名
style CSS 字符串 全局内联样式
backgroundColor 颜色值 全局背景色
backgroundImage URL 全局背景图

提示：header / footer 不会在标题页（第一页）显示，如果需要在标题页也显示，在标题页下方使用指令覆盖。

文件中（Marp Directive）速查

Directive（指令）用来控制单页或当前页之后的行为。分为全局指令和局部指令。

全局指令（Front Matter 中设置）↑ 见上表

局部指令（HTML 注释中设置）

写在 --- 分页后的上一页或当前页内容上方：

指令 示例 作用
<!-- _class: --> <!-- _class: lead --> 当前页使用指定类名
<!-- _backgroundColor: --> <!-- _backgroundColor: #123456 --> 当前页背景色
<!-- _backgroundImage: --> <!-- _backgroundImage: url(bg.jpg) --> 当前页背景图
<!-- _paginate: --> <!-- _paginate: skip --> 跳过当前页页码
<!-- _header: --> <!-- _header: 章节标题 --> 覆盖当前页及之后的页眉
<!-- _footer: --> <!-- _footer: 定制页脚 --> 覆盖当前页及之后的页脚
<!-- _style: --> <!-- _style: h2 { color: red; } --> 当前页内联样式
<!-- color: --> <!-- color: #333 --> 设置文字颜色

指令中的下划线前缀 _ 表示该指令仅影响当前页（而非全局）。

实时预览

安装插件后，打开 .md 文件，单击右上角 “Open Preview” 按钮，即可在右侧看到实时渲染的幻灯片效果。

导出

Marp for VS Code 支持三种导出格式：

导出 PDF

  1. 打开 Markdown 文件
  2. 点击右上角 Marp 菜单 → “Export Slide Deck…”
  3. 选择 PDF
  4. 保存

导出 PPTX

  1. 同样在 “Export Slide Deck…” 中选择 PowerPoint (PPTX)
  2. Marp 会生成标准 .pptx 文件，可在 PowerPoint / Keynote 中进一步编辑

导出 HTML

  1. 选择 HTML
  2. 生成一个单页 HTML 文件，双击即可在浏览器中展示，适合分享

快捷键

操作 快捷键
预览 Ctrl+K V
导出菜单 Ctrl+Shift+P → “Marp: Export”
切换主题 Ctrl+Shift+P → “Marp: Change Theme”
