小海龟例程

sudo apt-get install ros-noetic-turtle-tf
roslaunch turtle_tf turtle_tf_demo.launch
20.04 noetic 版本运行失败解决办法：

  1. cd /usr/bin/
  2. sudo rm -r python
  3. sudo cp python3 python

rosrun tf view_frames

运行rosrun tf view_frames报错解决方法
第89行代码m = r.search
报错如下

到/opt/ros/noetic/lib/tf目录下运行
sudo chmod 777 view_frames
将89行的m = r.search(vstr)更改为m = r.search(vstr.decode('utf-8'))

运行rosrun tf view_frames后生成一个pdf


三个坐标系

  1. world
  2. turtle1
  3. turtle2

rosrun tf tf_echo

rosrun tf tf_echo turtle1 turtle2，查看turtle1，turtle2两个坐标之间的位置关系

rosrun rviz

rosrun rviz rviz -d 'rospack find turtle_tf' /rviz/turtle_rviz.rviz

Fixed Frame 改成 world；然后左下角Add–>TF
