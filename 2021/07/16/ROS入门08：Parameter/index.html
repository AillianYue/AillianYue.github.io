ROS入门08：Parameter

|总字数:976|阅读时长:4分钟|浏览量:

命令行使用

小乌龟为例子
rosparam + 回车


rosparam list：列出当前参数
rosparam get param_key：显示某个参数值
rosparam set param_key param_value：设置某个参数值
rosparam dump file_name：保存参数到文件
rosparam load file_name：从文件读取参数
rosparam delete param_key：删除参数

rosparam list：列出当前参数

运行小海龟看小海龟的参数
roscore
rosrun turtlesim turtlesim_node
rosparam list

rosparam get：显示某个参数值

比如得到背景颜色
rosparam get /turtlesim/background_r
rosparam get /turtlesim/background_g
rosparam get /turtlesim/background_b

rosparam set：设置某个参数值

修改背景颜色
rosparam set /turtlesim/background_r 255
这时颜色并没有改变，运行rosservice call /clear "{}"

rosparam dump：保存参数到文件

将参数保存成**.yaml**文件
rosparam dump param.yaml

rosparam load：从文件读取参数

先修改param.yaml的参数如图


然后从文件读取参数
rosparam load param.yaml
刷新一下rosservice call /clear "{}"，小乌龟的颜色就变成（0，0，0）了

rosparam delete param_key：删除参数

rosparam delete /turtlesim/background_b
此时查看rosparam list，/turtlesim/background_b消失了


rosparam set /turtlesim/background_b 可以重新设置参数

创建功能包

cd ~/catkin_ws/src
catkin_create_pkg learning _parameter roscpp rospy std_srvs

  • get函数获取参数
